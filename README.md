Skill Swap is a mobile UI/UX design project focused on enabling users to exchange skills and items within a community through a simple, trust-based swapping experience.

Objective
To design an intuitive and modern mobile application that supports category-based browsing, detailed product descriptions, wishlist functionality, and a chat-based swap negotiation flow.

 Role
UI/UX Designer  
- User flow design  
- Wireframing  
- High-fidelity UI design  
- Interactive prototyping  

Tools Used
- Figma

Key Features
- Category-wise browsing
- Product detail pages
- Wishlist functionality
- Skill/item listing flow
- Request swap with chat
- Swap confirmation flow
- Profile and settings pages
